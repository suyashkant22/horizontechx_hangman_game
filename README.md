# horizontechx_hangman_game
A simple, text-based Hangman game built in Python where players have 6 attempts to guess a secret tech-related word.This project was created to demonstrate fundamental Python programming concepts such as loops, conditional statements, lists, strings, and the random module.  
Features
5 predefined tech-related words to guess.  
Limits the player to 6 incorrect guesses.  
Input validation (ensures the user types a single alphabetical letter).  
Tracks and displays previously guessed letters.
